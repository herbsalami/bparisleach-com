Forever Show is a side project on which I am currently working; it is a music streaming app for 1-hour mixes. My goal for this project was to simulate the experience of a 24-hour internet radio station without having to constantly book/stream live sessions. To achieve this, the server will select the mix that should be playing when a given user requests it, in accordance with a cycling schedule of mixes. The server then stashes the wav file's header, and begins reading the file from the timestamp at which the user would be served the audio, had it begun at the hour mark. Stack employed: HTML5, CSS, Javascript, NodeJS, Express.

Tracker-Tracker was a side project I worked on in order to pull data from private music trackers and present them to a client using graphs. A friend had a Raspberry PI set up gathering data and storing it in a MariaSQL database, and my server would query it and send it to the client via a Node/Express server. Stack employed: HTML, CSS, Javascript, ChartJS, NodeJS, Express, MariaSQL.
